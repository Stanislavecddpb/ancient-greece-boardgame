# Cyclades (браузерная версия)

Цифровая реализация настольной игры про борьбу за господство над островами
Эгейского моря. Сделана, чтобы играть с друзьями по сети.

## Цель игры

Первым собрать **2 Метрополии**. Метрополию можно получить тремя способами:

- построить все 4 типа зданий (Порт, Крепость, Храм, Университет);
- собрать 4 Философов;
- захватить чужую Метрополию в бою.

Ключевая механика — **скрытый аукцион** за расположение богов. Каждый цикл боги
(Арес, Посейдон, Зевс, Афина и Аполлон) дают разные действия, и игроки тайно
торгуются золотом за право выбрать нужного бога первым.

### Доход

Рога изобилия лежат на клетках доски. Рог **на воде** даёт золото тому, чей флот
стоит на этой клетке; рог **на суше** — владельцу острова. На доске на 4 игроков —
6 рогов на воде по краям плюс рога на суше (на части островов по два).

## Структура проекта

Монорепозиторий (npm workspaces) на базе **boardgame.io**:

- `packages/engine` — чистая игровая логика на TypeScript (правила, доска, бой).
- `packages/server` — авторитарный сервер на boardgame.io: комнаты, синхронизация.
- `packages/client` — клиент на React + Vite: SVG-доска (вид сверху), интерфейс.

## Запуск (разработка)

```bash
npm install
npm run dev:server   # сервер на :3001
npm run dev:client   # клиент на :5173
```

Открыть `http://localhost:5173`:
- **Создать игру** — получите ссылку, отправьте друзьям, играйте по сети.
- **Локальная игра** (`#/local`) — хотсит за одним компьютером.

## Тесты

```bash
npm test   # vitest, движок правил
```

## Статус и дорожная карта

Цель — **доведённая до конца игра**, а не прототип.

**Готово:**
- Цикл игры: доход → аукцион богов (с вытеснением ставок, жрецы-скидки) → действия.
- Действия богов: наём войск/флота/жрецов/философов, постройка зданий.
- Образование Метрополий (4 здания / 4 философа), условие победы.
- Доска на 4 игроков по точным координатам, рога изобилия, доход с них.
- Сетевая игра по ссылке (лобби, комнаты, переподключение) и локальный хотсит.
- Визуал: круглое поле с рельефными островами, фишки, карты богов.
- Боевая математика на костях (грани 0,0,1,1,2,3) с бонусом защиты.

**В работе / далее:**
1. **Движение и бой.** Перемещение флота по морским клеткам (до 3) и войск между
   островами по «мосту» из своих кораблей; бой с **отступлением**; захват островов
   и Метрополий; бонусы крепости/порта.
2. **Мифические существа.** Колода из 3 видимых существ за цикл, покупка за действие
   бога (Храм снижает цену): Кракен, Минотавр, Медуза, Полифем, Хирон, Пегас, Гиганты,
   Гарпии и т.д.
3. **Аполлон.** Маркер процветания вместо разовой выдачи золота.
4. **Карты на 2, 3 и 5 игроков** (сейчас точная — только на 4).
5. **UI боя и ходов:** показ фишек на карте, диалог боя/отступления, лог событий,
   индикаторы фаз, подсказки доступных действий.
6. **Развёртывание** сервера, чтобы играть с друзьями за пределами локальной сети.

Вне ближайших планов: дополнения (Hades, Titans, Monuments).

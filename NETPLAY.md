# Игра по сети с друзьями

Деплой не нужен. Ты запускаешь игру у себя, отдаёшь друзьям ссылку через туннель.
Клиент и сервер работают на **одном порту** (3001), поэтому туннель нужен один.

## 1. Запустить игру у себя

```
npm install        # один раз
npm run play       # собирает клиент и поднимает сервер на http://localhost:3001
```

Проверь, что открывается `http://localhost:3001` — это и есть игра.

> Менять что-то в коде на ходу не получится: `play` отдаёт собранную версию.
> Пересобрать после правок — снова `npm run play`.

## 2. Открыть наружу туннелем

Твой компьютер должен оставаться включённым, пока идёт партия.

### Вариант А — Cloudflare Tunnel (бесплатно, без регистрации)

```
# поставить cloudflared (один раз), затем:
cloudflared tunnel --url http://localhost:3001
```

Команда выдаст публичный адрес вида `https://random-words.trycloudflare.com` —
это и есть ссылка на игру.

### Вариант Б — ngrok

```
ngrok http 3001
```

Возьми из вывода адрес `https://....ngrok-free.app`.

## 3. Играть

1. Открой публичную ссылку сам → «Создать игру» → выбери число игроков.
2. Нажми «Скопировать ссылку» и отправь друзьям — она уже с публичным адресом
   и кодом комнаты.
3. Друзья открывают ссылку и занимают места. Когда все в сборе — игра стартует.

Состояние партии хранится на сервере: если вкладка закрылась, можно вернуться
по той же ссылке и продолжить.

## Заметки

- **Локальная сеть (без туннеля):** если все в одном Wi-Fi, друзьям хватит
  `http://ТВОЙ_LAN_IP:3001` (узнать IP — `ipconfig`). Разреши порт в брандмауэре.
- **Туман войны** сейчас прячется на стороне клиента. Для друзей этого достаточно,
  но технически скрытые данные есть в сетевом трафике. Если нужно строго —
  вынесем фильтрацию на сервер отдельной задачей.
- **Хотсит** (все за одним компом) по-прежнему доступен: ссылка `/#/local`.
